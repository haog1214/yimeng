# 宜盟企業官網開發筆記

## 設計決策

### 色彩系統
- **主色** - 紅色 (#E63946)：來自設計圖的紅色邊框，用於強調、按鈕與互動元素
- **背景** - 白色與淺灰色 (#F5F5F5)：提供清潔、專業的視覺效果
- **文字** - 深灰色 (#2C3E50)：確保可讀性與專業感
- **強調背景** - 深灰色 (#1A1A1A)：用於聯絡區塊與頁尾

### 排版系統
- **標題** - Playfair Display (優雅、高端感)
- **副標題** - Poppins Bold (現代、有力)
- **正文** - Inter (清晰易讀)

### 佈局原則
- 使用非中心對稱的排列，創造動感與視覺興趣
- 紅色邊框卡片用於突出重點內容
- 充分利用負空間，避免擁擠感
- 響應式設計，適配各種螢幕尺寸

## 實現的區塊

1. **導航列** - 粘性導航，包含公司名稱與菜單連結
2. **英雄區** - 全寬灰底區域（待補圖片）
3. **關於宜盟** - 左側大標題 + 右側品牌介紹
4. **產品應用** - 左側圖片 + 右側四個應用類別卡片
5. **技術特色** - 射出成型與超薄模具成型的介紹
6. **聯絡我們** - 深灰背景區塊，包含聯絡方式與按鈕
7. **頁尾** - 公司資訊與營業時間

## 待補充項目

- 所有灰底區域需要補上實際圖片
- 聯絡表單功能（需要後端支援）
- 可選：添加動畫效果與過渡效果
- 可選：添加更多互動元素

## 技術棧

- React 19 + TypeScript
- Tailwind CSS 4
- shadcn/ui 元件庫
- Wouter 路由
